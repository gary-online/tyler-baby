#### Image Guidelines

- **Format**: JPG for photos (compress to ~200KB per image for optimal loading)
- **Quality**: High resolution but optimized for web
- **Consistency**: Similar lighting and color grading across all images
- **Branding**: Professional, elegant aesthetic matching the rose gold theme

- **Format**: MP4 with H.264 codec (most compatible)
- **Length**: Keep under 30 seconds for autoplay content
- **Quality**: Balance between quality and file size
- **Compression**: Use HandBrake or similar tool to compress
- **Sound**: Videos will autoplay muted, but include audio for user control
- **Content**: Professional, smooth footage with good lighting
- **Branding**: Consider adding subtle watermark or logo

## How to Add Your Assets

### For Images

1. Place your images in the `images/` folder
2. Name them exactly as specified above (case-sensitive)
3. Ensure they are optimized for web (use tools like TinyPNG or ImageOptim)
4. Recommended formats: JPG for photos, PNG for graphics with transparency

### For Videos

1. Place your videos in the `videos/` folder
2. Name them exactly as specified above
3. Ensure they are in MP4 format with H.264 codec
4. Test autoplay functionality on various devices

## Best Practices

1. **Consistency**: Use similar filters/editing across all photos
2. **Quality**: Never use blurry or pixelated images
3. **Loading Speed**: Compress all assets without visible quality loss
4. **Accessibility**: Use descriptive alt text (already included in HTML)
5. **Backup**: Keep original high-res versions separate
6. **Rights**: Ensure you have permission for all client photos
7. **Updates**: Refresh gallery regularly with new transformations
